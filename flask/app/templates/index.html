v1 limitations (I'll try to add these to v2):

  1. Not the creators voice in a different language
  2. Only male voices for now
  3. limited output languages
  4. 1 GPU
  5. 10 min video limit

I came across an ML paper which I couldn't understand. The only resource I found was a YouTube video in Korean.
Since I couldn't understand it, I tried researching some more.
But after failing to find any other resource except the 1 Korean video, I figured I'll just convert it myself.
Once I did that, I figured there might be others like me looking to translate obscure videos, so I built this extension out.
It will be a little slow as I'm only using one GPU to prevent exorbitant costs to keep this free for everyone. Enjoy!
